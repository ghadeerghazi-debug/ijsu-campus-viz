# 🌞 محاكاة الطاقة الشمسية - الحرم الجامعي
## Imam Al-Sadiq University Campus Solar Visualization

تطبيق ويب تفاعلي يعرض:
1. **خريطة الحرم الجامعي** - 14 مبنى مع تفاصيل كاملة لكل سطح
2. **محاكاة تدفق الطاقة** - عرض حي على مدار 24 ساعة

## ميزات التطبيق

### 🗺️ خريطة الحرم الجامعي (Campus Map)
- خريطة طوبولوجية تعرض جميع مباني الجامعة الـ14
- ألوان تعكس مستوى الجاهزية للتركيب (ممتاز / جيد / متوسط)
- نقر على أي مبنى لعرض:
  - **صورة السطح الفعلية** (من المسح الميداني)
  - **لقطة جوية من Google Earth**
  - مساحة السطح بالأمتار المربعة
  - الإمكانية الكهربائية بالكيلوواط
  - الطلب الكهربائي التقديري
  - عدد الألواح الشمسية المقترحة
  - ملاحظات تركيب خاصة
- تصفية حسب الجاهزية
- ترتيب المباني تلقائياً حسب الإمكانية

### ⚡ محاكاة تدفق الطاقة (Energy Flow)
- شريط زمني تفاعلي 24 ساعة
- وضع التشغيل التلقائي
- ثلاث حالات طقس: مشمس / غائم / ليل
- مؤشرات حية:
  - الإنتاج الحالي
  - الاستهلاك الحالي
  - حالة البطاريات (شحن / تفريغ / توازن)
  - الصافي (فائض / عجز)
- مخطط متحرك يُظهر تدفق الطاقة:
  - الشمس → الألواح → العاكس → البطاريات → المباني
  - الشبكة الوطنية كاحتياطي ليلاً
- منحنى يومي يقارن الإنتاج بالاستهلاك

## كيفية التشغيل

```bash
# 1. ثبّت المتطلبات
npm install

# 2. شغّل التطبيق
npm start
```

## للنشر على Vercel

1. ادفع المجلد إلى GitHub
2. اربط حسابك على Vercel.com بـ GitHub
3. انقر Import → اختر المستودع → Deploy
4. ستحصلين على رابط حي خلال 3 دقائق

## بيانات المباني (الإجمالي)

- **عدد المباني:** 14
- **إجمالي مساحة الأسطح:** 9,659 م²
- **الإمكانية الكهربائية النظرية:** ~772 ك.و
- **الحاجة الفعلية للجامعة:** 440 ك.و
- **النتيجة:** يكفي تركيب 5-6 مباني فقط لتغطية الحاجة الكاملة
